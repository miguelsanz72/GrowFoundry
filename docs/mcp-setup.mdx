## Overview

The GrowFoundry MCP server gives your AI coding assistant direct access to your backend: database queries, schema management, storage, and more.

Follow the instructions below for your AI client. Don't see your tool? Browse the [agent directory](https://growfoundry.dev/agents) for the full list of tested integrations.

## Prerequisites

- An **AI coding assistant** (Cursor, Claude Code, GitHub Copilot, etc.)
- An **GrowFoundry project** - [create one here](https://growfoundry.dev) if you haven't already

## Remote MCP

The fastest way to get started is with Remote MCP. A single command attempts installation and will try to handle authentication and project binding, but it may still prompt you for manual authentication or project binding steps if the automatic flow cannot complete.

Run the following command in your terminal:

```bash
npx add-mcp https://mcp.growfoundry.dev/mcp
```

<Note>
  The `add-mcp` command works for most coding agents, but not all. Currently it supports: 
  Claude Code, Claude Desktop, Codex, Cursor, Gemini CLI, Goose, GitHub Copilot, OpenCode, VS Code, Zed.
</Note>

After running the command, the system will automatically generate a configuration file for your coding agent.

### Connect and Bind Project

Some MCP clients will automatically prompt you to login during setup, while others may require manual authentication or project binding steps if the automatic flow cannot complete. Either way, authentication will open a browser window where you can login to your GrowFoundry account and grant organization/project access to the MCP client.

<AccordionGroup>
  <Accordion title="Cursor">
    You can find the configuration file in `.cursor/mcp.json` as below:

    ```json
    {
      "mcpServers": {
        "growfoundry": {
          "url": "https://mcp.growfoundry.dev/mcp"
        }
      }
    }
    ```

    In menu **Preferences → Cursor Settings → Tools & MCP**, you should see GrowFoundry MCP server is added successfully.
    GrowFoundry MCP server is on the status of "Needs authentication", you can click the `connect` button to authenticate.
  </Accordion>

  <Accordion title="Claude Code">
    After installation, you will find the configuration file in `.mcp.json` as below:
    ```json
    {
      "mcpServers": {
        "growfoundry": {
          "type": "http",
          "url": "https://mcp.growfoundry.dev/mcp"
        }
      }
    }
    ```
    Then in a regular terminal (not the IDE extension) run:
    ```sh
    claude /mcp
    ```
    Select the "growfoundry" server, then "Authenticate" to begin the authentication flow.

  </Accordion>

  <Accordion title="GitHub Copilot">
    After installation, you will find the configuration file in `.vscode/mcp.json` as below:
    ```json
    {
      "servers": {
        "growfoundry": {
          "type": "http",
          "url": "https://mcp.growfoundry.dev/mcp"
        }
      }
    }
    ```
  </Accordion>

  <Accordion title="Google Antigravity">
    Add this configuration to `~/.gemini/antigravity/mcp_config.json`:
    ```json
    {
      "mcpServers": {
        "growfoundry": {
          "serverUrl": "https://mcp.growfoundry.dev/mcp"
        }
      }
    }
    ```
    Then restart Antigravity, it will auto authenticate.
  </Accordion>

  <Accordion title="Gemini">
    After installation, you will find the configuration file in `.gemini/settings.json` as below:
    ```json
    {
      "mcpServers": {
        "growfoundry": {
          "type": "http",
          "url": "https://mcp.growfoundry.dev/mcp"
        }
      }
    }
    ```
    Start the Gemini CLI and run the following command to authenticate the server:
    ```sh
    /mcp auth growfoundry
    ```
  </Accordion>

  <Accordion title="Codex">
    <Note>
      If you are using Codex for the first time, you may need to enable the rmcp feature. To do so, add the following into your `~/.codex/config.toml` file:
      ```toml
      [beta]
      rmcp = true
      ```
    </Note>
    After installation, you will find the configuration file in `.codex/config.toml` as below:
    ```toml
    [mcp_servers.growfoundry]
    type = "http"
    url = "https://mcp.growfoundry.dev/mcp"
    ```
    Then authenticate:
    ```sh
    codex mcp login growfoundry
    ```
    Finally, run /mcp inside Codex to verify authentication.

    Alternatively, you can run the following command to install the MCP server:
    ```sh
    codex mcp add growfoundry --url https://mcp.growfoundry.dev/mcp
    ```
  </Accordion>

  <Accordion title="Cline">
    You can install the GrowFoundry MCP server to your **Cline** by running the following command in your terminal:
    ```bash
    npx -y @smithery/cli mcp add devel/growfoundry --client cline
    ```
    Alternatively, you can follow the official [Cline documentation](https://docs.cline.bot/mcp/connecting-to-a-remote-server) to add the GrowFoundry MCP server with parameters:
    - Server Name: `growfoundry`
    - Server URL: `https://mcp.growfoundry.dev/mcp`
    - Transport Type: `Streamable HTTP`

    After adding the server, you’ll see an error message asking to authenticate. Click the “Authenticate” button that appears to authenticate.

  </Accordion>

  <Accordion title="Windsurf">
    Add this configuration to `~/.codeium/windsurf/mcp_config.json`:
    ```json
    {
      "mcpServers": {
        "growfoundry": {
          "command": "npx",
          "args": [
            "-y",
            "mcp-remote",
            "https://mcp.growfoundry.dev/mcp"
          ]
        }
      }
    }
    ```
    Windsurf does not currently support remote MCP servers over HTTP transport. You need to use the mcp-remote package as a proxy.
  </Accordion>

  <Accordion title="Roo Code">
    You can install the GrowFoundry MCP server to your **Roo Code** by running the following command in your terminal:
    ```bash
    npx -y @smithery/cli mcp add devel/growfoundry --client roocode
    ```
  </Accordion>

  <Accordion title="OpenCode">
    After installation, you will find the configuration file in `./opencode.json` as below:
    ```json
    {
      "mcp": {
        "growfoundry": {
          "type": "remote",
          "url": "https://mcp.growfoundry.dev/mcp",
          "enabled": true
        }
      }
    }
    ```
    Then run the following command to authenticate:
    ```sh
    opencode mcp auth growfoundry
    ```
    This will open your browser to complete the OAuth authentication flow.
  </Accordion>
