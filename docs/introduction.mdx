GrowFoundry gives your coding agent Database, Authentication, Storage, Realtime, Edge Functions, Sites, and a Model Gateway to ship full-stack apps end to end.

## Connect first

Required setup. Pick one path to link your environment to GrowFoundry before you start building. Start with the CLI for a universal terminal-based setup; add MCP if your editor speaks it (Claude Code, Cursor, Windsurf, Codex). You can run both side by side.

<CardGroup cols={2}>
  <Card title="Connect via CLI" icon="terminal" href="/quickstart">
    `npx @growfoundry/cli login` and link a project. Works in any terminal, MCP not required.
  </Card>
  <Card title="Connect via MCP" icon="plug" href="/mcp-setup">
    Wire your editor's MCP client so the agent gets live schemas, logs, and project context.
  </Card>
</CardGroup>

Looking for instructions for a specific tool? Browse the [agent directory](https://growfoundry.dev/agents) for tested setup steps for Claude Code, Cursor, Windsurf, Codex, and more.
